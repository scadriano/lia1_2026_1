# LIA1 - Laboratório de Inovação e Automação 1

## Sobre o Repositório

Este repositório contém as entregas de atividades, projetos e exercícios desenvolvidos durante a disciplina de Laboratório de Inovação e Automação 1 (LIA1) no período letivo de 2026/1.

## Estrutura

O repositório está organizado em pastas individuais para cada aluno, onde são armazenados os trabalhos e entregas realizados ao longo do semestre.

## Disciplina

- **Curso:** Laboratório de Inovação e Automação 1
- **Período:** 2026/1
- **Objetivo:** Desenvolvimento de competências em automação, programação e inovação tecnológica
